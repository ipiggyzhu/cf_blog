## 🛠?常用命令速查

```bash
# 初始化仓?
git init

# 克隆远程仓库
git clone <url>

# 查看状?
git status

# 添加文件
git add .

# 提交更改
git commit -m "提交信息"

# 推送到远程
git push origin main

# 拉取更新
git pull origin main

# 创建分支
git branch <branch-name>

# 切换分支
git checkout <branch-name>

# 合并分支
git merge <branch-name>
```
